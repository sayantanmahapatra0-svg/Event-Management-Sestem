Welcome To Event Management

The Event Management Website is a web-based platform designed to facilitate event planning and participation. It provides two distinct user roles: Customers (event attendees) and Organizers (event creators). Each user type has a separate login page, unique profile ID, and dedicated functionalities to view profiles and apply (customers apply to attend events, organizers apply to create events). The backend is responsible for handling user authentication, data storage, profile management, event management, and application processing.
